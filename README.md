This is a lightweight Tampermonkey userscript that provides a small floating interface
to quickly open recent worldwide YouTube search results using custom keywords.

## Features
- Floating draggable interface
- Custom keyword input
- Worldwide search results
- Recent uploads filter
- No data collection
- No automation
- No API usage

## Requirements
- Browser with Tampermonkey installed

## Usage
1. Install Tampermonkey
2. Install the userscript
3. Open YouTube
4. Enter keywords and open recent worldwide search
